Get Started

The platform consists of two apps: 5ervice and 5ervice Business.

git clone https://github.com/lorcanblake/Service.git
        cd Service
        npm i
        ng serve --o
        
git clone https://github.com/lorcanblake/Service-business.git
        cd Service-business
        npm i
        ng serve --o

Prototyping

Once you've grabbed Etta, prototyping is as easy as pie:

<!DOCTYPE html>
        <html lang="en">
        <title> </title>
        <meta name="viewport" content="width=device-width, initial-scale=1">
        <link rel="stylesheet" href="/path/to/etta.min.css">
        <body>
        
        </body>
        </html>
